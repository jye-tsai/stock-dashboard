技術與安全

🛠️ 怎麼做的

  • 單一 HTML 檔:HTML + CSS + JS 全內嵌,好部署、好離線
  • Chart.js 畫所有圖表,配色隨主題變
  • GitHub Pages 靜態託管,資料存 data.json
  • GitHub Action + Cloudflare Worker 排程更新市價
  • PWA:manifest + Service Worker 離線快取

🔒 安全性

  • 資料防竄改靠 GitHub 寫入權限 / token,沒 token 誰都存不回
  • 密碼用 PBKDF2-SHA256 + 隨機鹽(前端遮罩用)
  • token 只存你裝置瀏覽器與 Cloudflare Secret,不進 repo
  • data.json 在公開 repo,金額資訊留意隱私
